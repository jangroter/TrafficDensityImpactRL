IC: Initial Condition

Load a new initial condition, or re-load the current initial condition. When called without arguments a file dialog is shown. When called as:

IC IC

the current initial condition is re-started.

Usage:

IC [IC/filename]

Arguments:

Name Type Required Description
IC string no 'IC IC' reloads the current initial condition.
filename string no Load the initial condition in file 'filename'
